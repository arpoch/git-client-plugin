Set the git username / password credential for HTTP and HTTPS protocols.

Shell example 

withCredentials([GitUsernamePassword('my-credentials-id')]) {
  sh 'git fetch --all'
}

Batch example 

withCredentials([GitUsernamePassword('my-credentials-id')]) {
  bat 'git submodule update --init --recursive'
}

Powershell example 

withCredentials([GitUsernamePassword('my-credentials-id')]) {
  powershell 'git push'
}
